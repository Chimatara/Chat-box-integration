# Lemlist Integration

This project is a small landing page for a Nigerian catering business called Tara's Kitchen.

## What it is for

- This is strictly for learning purpose
- Showcases a catering service with sections for About, Menu, Why Choose Us, and Contact.
- Provides a responsive marketing page for event catering services in Lagos, Abuja, and Port Harcourt.
- Lets visitors send enquiries through a contact form that uses EmailJS.

## What it does

- Displays a hero section with a booking call to action.
- Presents the company's story, statistics, testimonials, sample menu, and service benefits.
- Includes a mobile-friendly navigation menu.
- Filters sample menu items by category.
- Sends enquiry form submissions via EmailJS.

## Who it solves problems for

- Small catering or hospitality businesses that need a simple web presence.
- Event planners and customers looking for Nigerian catering services.
- Developers who want a starter template for a Vue-powered landing page with a contact form and email integration.

## Files

- `index.html` — main page markup and Vue app mounting point.
- `index.js` — Vue app logic, menu filtering, form validation, and EmailJS submission.
- `style.css` — page styling and responsive layout.
- `Assets/` — project assets such as logo images.

## Notes

- The site uses Vue 3 via CDN and EmailJS for form submission.
- Update the EmailJS service/template IDs in `index.js` if you connect your own EmailJS account.
